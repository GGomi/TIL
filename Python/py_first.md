## 2. 문자열 자료형
- 문자열 만드는 방법은 총 4가지로 나뉜다.
```
1. 큰따옴표
2. 작은 따옴표
3. 큰따옴표 3개 연속
4. 작은따옴표 3개 연속
```
- 이유는 큰따옴표나 작은따옴표를 문자열에 포함시키고 싶을때 감싸는 기호가 겹치면 인식시키기 힘들기때문에
- 문자열 더하기는 여느 언어들과 차이없었는데 * 개념은 처음 보는 개념이었다.
```
#문자열 곱하기

a*2
'pythonpython'
```
