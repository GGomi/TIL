- 문자열 대입 및 포매팅 방법
```
#문자열 포매팅
a = "My apple %d" % 3
=> %d에 3 이라는 정수가 대입됨.

#포맷 함수를 이용한 대입
name = "jeongmin"
a = "{0} is my name".format(name)
"jeongmin is my name"
```

- 문자열 정렬
```
#문자열 왼쪽정렬
"{0:<10}".format("hi")
- hi라는 문자열을 왼쪽정렬 시키고 나머지 부분을 공백으로 채워서 10자리로 맞춰준다.

#문자열 가운데 정렬
"{0^10}".format("hi")
- 가운데 hi를 위치시키고 왼쪽오른쪽을 공백으로 채워서 10자리 맞춤

#문자열 오른쪽 정렬
"{0:>10}".format("hi")
- hi라는 문자열을 오른쪽정렬 시키고 나머지 부분을 공백으로 채워서 10자리로 맞춰준다.

- 정렬 기호 바로앞에 다른 문자를 넣어주면 공백대신 해당하는 문자로 자동으로 채워진다.
"{0:!>10}".format("hi")
>>> "!!!!!!!!hi"
```
