## 1. General Rules
- 문서 생성은 [GFM (Github Flavored Markdown)](https://help.github.com/articles/github-flavored-markdown/) 을 사용한다. (확장자 `.md`)
- 언어나 기술명으로 폴더를 만든다. (root에 문서를 만들지 않는다. 오직, md파일과 폴더만)

## 2. Naming Rules
- 파일명은 영어로.
- 알아보기쉽고, 직관적으로 만든다.
- 단어구분은 `_`로 한다
